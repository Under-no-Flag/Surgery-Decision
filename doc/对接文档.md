# 术中获得性压力性损伤风险预警及决策辅助系统

已收集需求
前端：微信H5页面
要有后台页面，数据库

不参与计算的输入项  （靠前）
住院号 （ 用户自己输入，不参与计算）
术后皮肤情况（压红、一期压力性损伤、二期压力性损伤、其他，四选一）
用户输入选项
1.术中体位
1=中转体位，2=侧卧位，3=俯卧位，4=仰卧位，5=截石位
2.手术等级
1=四级，2=三级，3=一级或二级
3.手术方式
1=中转开腹，2=开腹，3=浅表或深部组织，4=微创(腔镜)
4.BMI
用户只填写体重(KG)、身高 (米)（系统计算BMI    BMI=体重(KG)/身高(米) 的平方  ）
5.麻醉诱导时低低温
0=≥36°C     1=<36°C
6.葡萄糖异常
0= 3.88-6.11     1=<3.88 或>6.11
7.白蛋白异常
0=40-55，1=<40或>55
8.预计手术时间(分钟)


计算公式
logit(P) =5.3197+ -0.4604 x术中体位+ -0.6223x 手术等级+ -0.3634 x 手术方式+ -0.0897 x BMI+1.0467 x 诱导低体温+0.6037x白蛋白异常+0.6432x葡萄糖异常+0.0019x手术时间
P= 1/ (1 + exp(-logit(P)))

建议文本
1.＜30%
低风险，建议（1）保持受压部位皮肤清洁、干燥；（2）使用具有记忆海绵手术床垫，弹性和支撑度良好；（3）规范安置手术体位，保持肢体、躯干处于功能位，避免过度牵拉增加剪切力；（4）使用棉质/海绵/凝胶/流体等材质体位垫托起肢体；（5）采用综合保温措施，维持核心体温在正常范围内；（6）遵医嘱选择输注液体或血制品类别，维持循环稳定；（7）术中调整或变换手术体位时，受压部位增加棉质/海绵/凝胶/流体等体位垫进行减压预防。

2.30%-48%
中风险，建议（1）保持受压部位皮肤清洁、干燥；（2）使用具有记忆海绵手术床垫，弹性和支撑度良好；（3）规范安置手术体位，保持肢体、躯干处于功能位，避免过度牵拉增加剪切力；（4）使用棉质/海绵/凝胶/流体等材质体位垫托起肢体；（5）体位安置前在手术床上使用凝胶/流体等材质体位垫，受压部位皮肤应采用预防性敷料；（6）采用综合保温措施，维持核心体温在正常范围内；（7）遵医嘱选择输注液体或血制品类别，维持循环稳定；（8）术中调整或变换手术体位时，受压部位增加棉质/海绵/凝胶/流体等体位垫进行减压预防。

3.＞48%
高风险，建议（1）保持受压部位皮肤清洁、干燥；（2）使用具有记忆海绵手术床垫，弹性和支撑度良好；（3）规范安置手术体位，保持肢体、躯干处于功能位，避免过度牵拉增加剪切力；（4）使用棉质/海绵/凝胶/流体等材质体位垫托起肢体；（5）体位安置前在手术床上使用凝胶/流体等材质体位垫，受压部位皮肤应采用预防性敷料；（6）采用综合保温措施，维持核心体温在正常范围内；（7）遵医嘱选择输注液体或血制品类别，维持循环稳定；（8）术中调整或变换手术体位时，受压部位增加棉质/海绵/凝胶/流体等体位垫进行减压预防；（9）在手术允许情况下，术中针对受压部位进行手术体位微调整。
