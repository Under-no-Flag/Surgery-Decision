## 核心授权流程

### 引导用户进入授权页
当用户访问你的 H5 页面时，检测到未登录，前端或后端将页面重定向到微信的授权 URL：

https://open.weixin.qq.com/connect/oauth2/authorize?appid=APPID&redirect_uri=REDIRECT_URI&response_type=code&scope=SCOPE&state=STATE#wechat_redirect


### 用户同意授权，获取 code
用户点击确认后，微信会跳转回你填写的 redirect_uri，并在 URL 参数中带上 code。

示例：http://yourdomain.com/index.html?code=CODE&state=STATE


### 换取 access_token （后端操作）
这一步必须在服务器端完成。通过 code 调用接口：
```
GET https://api.weixin.qq.com/sns/oauth2/access_token?appid=APPID&secret=SECRET&code=CODE&grant_type=authorization_code
```

### 拉取用户信息（后端操作）
如果 scope 为 snsapi_userinfo，可以使用上一步获得的 access_token 获取用户信息
```
GET https://api.weixin.qq.com/sns/userinfo?access_token=ACCESS_TOKEN&openid=OPENID&lang=zh_CN
```
